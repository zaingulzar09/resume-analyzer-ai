🚀 How to Get a FREE API Key (No Credit Card Required)

  1. Go to console.groq.com and sign up with Google/GitHub
  2. Navigate to API Keys in your dashboard
  3. Click Create API Key and copy your key (starts with gsk_)
  4. Create or edit the .env file in your application root directory
  5. Add your API key in the following format:
LLM_PROVIDER=groq
GROQ_API_KEY=gsk-your-api-key-here

Alternative providers: You can also use OpenAI (OPENAI_API_KEY) or Gemini (GEMINI_API_KEY) by changing LLM_PROVIDER.

After updating the .env file, restart the application for changes to take effect.
