# AI Resume Analyzer & Optimizer

AI-powered resume analysis and optimization using Groq's Llama 3.3.

## Features
- AI resume analysis with ATS scoring (0-100)
- Resume optimization with STAR method
- Job-specific keyword analysis
- User authentication & dashboard

## Setup
1. Install Python 3.8+
2. `pip install -r requirements.txt`
3. Create `.env` file with:
LLM_PROVIDER=groq
GROQ_API_KEY=your-key-here
4. `python app.py`
5. Open `http://127.0.0.1:5000`
